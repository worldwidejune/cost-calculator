# 国际异动成本计算器

在线工具，用于测算和对比国际差旅、LTBS、国际派驻三种异动模式的增加成本。

## 功能

- 🛫 **国际差旅**成本测算（交通 + 酒店 + 补贴）
- 🔄 **LTBS**（Long-Term Business Stay）成本估算
- 🏠 **国际派驻**成本测算（补贴 + 税负平衡）
- 📊 三种模式**横向对比**，一目了然

## 使用方式

直接访问 GitHub Pages 链接即可使用，无需安装。

## 数据说明

- 交通费用均为参考数据，实际费用请参考差旅预订平台的价格
- 汇率默认 USD/CNY = 7.25，支持在线获取实时汇率
- 酒店、补贴标准基于公司政策
